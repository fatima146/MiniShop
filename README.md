# MiniShop

React Native shop app gebouwd met Expo, TypeScript, TanStack Query en Redux Toolkit.

## Features

- Producten bekijken van DummyJSON API
- Product detail met afbeelding vergroten
- Winkelwagen met +/- en verwijderen
- Light/Dark theme toggle
- Profile met cart summary

## Tech Stack

- Expo Router (file-based navigation)
- TypeScript
- TanStack Query (data fetching)
- Redux Toolkit (state management)

## Installatie

```bash
npm install
npx expo start
```
